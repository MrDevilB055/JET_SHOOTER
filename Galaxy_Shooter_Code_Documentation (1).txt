  Name                  Type       Purpose
  --------------------- ---------- ---------------------------------
  Player_health         Variable   Stores player’s health
  player_jetx           Variable   Player X position
  player_jety           Variable   Player Y position
  player_jetacc         Variable   Jet acceleration
  player_jetfriction    Variable   Slows jet gradually
  player_jetvelx        Variable   Horizontal velocity
  player_jetvely        Variable   Vertical velocity
  player_jet            Surface    Original player sprite
  playerjet             Surface    Scaled player sprite
  player_jet_surface    Rect       Collision box of player

  player_jet2_surface   Rect       Collision box for alternate jet
  jet_pos               Vector2    Stores jet position vector
  Player_Alive          Boolean    Whether player is alive
